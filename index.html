jquery-scroll-rotate v2 on Github

A super-tiny (<1KB, minified) jQuery (or Zepto) plugin to rotate elements by scrolling window
by @iqbalperkasa

Default usage

elem.scrollRotate();

Couterclockwise

elem.scrollRotate({
	clockwise: false
});

Without smooth transition

elem.scrollRotate({
	smooth: false
});

Faster speed

elem.scrollRotate({
	speed: 2
});

Fasterer

elem.scrollRotate({
	speed: 100
});
